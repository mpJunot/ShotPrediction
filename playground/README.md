# Streamlit Playground

This directory contains the Streamlit web application for the Basketball Trajectory Analyzer.

## Files

- `streamlit_app.py` - Main Streamlit application
- `styles.css` - Custom CSS styling for the application
- `run_streamlit.py` - Script to launch the Streamlit app

## Usage

### Run from playground directory

```bash
cd playground
python run_streamlit.py
```

Or directly with Streamlit:

```bash
cd playground
streamlit run streamlit_app.py
```

### Run from project root

```bash
python playground/run_streamlit.py
```
